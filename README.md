# GENESI LACTO T Site

Landing page comercial do GENESI LACTO T.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Publicação

O deploy é feito pelo GitHub Actions em `main`, publicando a pasta `dist` no GitHub Pages.

Domínio configurado:

```txt
lactot.genesi.com.br
```
